# HydroArray

A comprehensive hydrological data processing and modeling library for Python with continuous development.

## Features

- **Traditional Hydrological Models**: XinAnjiang, HyMOD, SAC-SMA, CREST
- **Machine Learning Models**: LSTM, ConvLSTM, U-Net, Diffusion
- **Flow Routing**: Linear Reservoir, Kinematic Wave, Muskingum
- **Data Assimilation**: Ensemble Kalman Filter, Particle Filter
- **Sensitivity Analysis**: Sobol' indices, Morris method
- **Model Calibration**: Genetic Algorithm, SCE-UA, Scipy optimizers
- **Ensemble Forecasting**: Multi-model and parameter ensembles
- **Data I/O**: Grid (ASC/GeoTIFF), Time series (CSV/Excel/NetCDF)

## Installation


```bash
git clone https://github.com/your-repo/HydroArray.git
cd HydroArray
pip install -e .
```
